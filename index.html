Architecture Case Studies

01

Big Data to Cloud Data Warehouse Migration

Problem

Legacy Big Data ecosystems at Citi were limiting data accessibility and platform scalability. The existing infrastructure created bottlenecks in enterprise data consumption and analytics.

Architecture Approach

Designed a phased migration to Starburst Enterprise as the cloud data warehouse layer, with API-first microservices providing unified data access. Introduced an API Gateway pattern with OAuth 2.0/SSO for secure, governed data access.

Starburst EnterpriseSpring BootKubernetesAmazon S3OAuth 2.0
Outcome: Significant improvement in platform scalability, data accessibility, and query performance across enterprise teams globally.
02

Real-Time Enterprise Platform with WebSocket

Problem

Enterprise workforce dashboards relied on polling-based data refresh, leading to stale data, poor user experience, and excessive server load under concurrent usage.

Architecture Approach

Implemented WebSocket-based event-driven communication between Angular frontends and backend microservices. Designed an event bus architecture for real-time data propagation with fallback REST APIs for non-real-time consumers.

WebSocketAngularSpring BootEvent-DrivenOpenShift
Outcome: Eliminated polling overhead, delivering live data updates with significantly reduced latency and improved user experience across enterprise dashboards.
03

Engineering Quality Through TDD/BDD & Contract Testing

Problem

Rapid microservices growth at Citi caused frequent integration defects and API contract violations between independently deployed services, slowing release velocity.

Architecture Approach

Established organization-wide engineering standards: TDD for unit coverage, BDD for acceptance criteria, and consumer-driven contract testing (Pact) to verify API contracts between microservices before deployment.

TDDBDDContract TestingPactCI/CDTekton
Outcome: Reduced integration defects, improved API reliability, and enabled confident independent service deployments at high velocity.
